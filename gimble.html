Gimble : An in-house SaaS Application

About The App

Peter Honey found a gap in the plumbing industry, it was nearly impossible for the plumber to get the best price out of the quotes received from the suppliers, thus came Gimble.
Gimble is a PHP web application that also communicates with Android and IPhone mobile applications. Its primary feature is to act as a bidding system for the suppliers to make bids on the request made by the plumber but it also works from the plumber's side only where the plumber can upload PDFs received from suppliers to be translated by the system as if the supplier had entered it in directly.
The mobile application also allows the user to search for the nearest suppliers based on the user's location.
